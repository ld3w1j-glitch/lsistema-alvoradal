# Favicon Alvorada

Ícone da aba configurado em `templates/base.html` e arquivos adicionados em `static/img/`.

Arquivos:

- `static/img/favicon.ico`
- `static/img/favicon.png`
- `static/img/favicon-32.png`
- `static/img/favicon-16.png`
- `static/img/apple-touch-icon.png`

Depois de subir no Railway, use `CTRL + F5` ou aba anônima para limpar cache.
